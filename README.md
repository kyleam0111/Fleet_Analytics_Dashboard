# CSX_Fleet_Analytics_Dashboard

This project analyzes key fleet performance indicators using real NYC Fleet Daily Service data. It focuses on KPIs that align with industry fleet operations, including:

### Core KPIs
- Fleet utilization rate
- Preventive maintenance (PM) compliance
- Repair cost per vehicle
- Availability and downtime trends
- Operational readiness metrics

### Tools
- Power BI
- Excel
- Real public fleet datasets (NYC Open Data)

### Purpose
This dashboard was designed to model the type of analytics used in fleet operations roles such as the Vehicles Analyst position at CSX. It demonstrates KPI creation, operational data analysis, and performance reporting.
